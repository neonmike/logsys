# 日志模块说明文档

## 项目规划
- 目标：在 Linux/Ubuntu 环境交付一个轻量级 C 语言日志库，支持控制台与文件输出、日志级别过滤、时间戳、本地/UTC时间、按大小轮转与备份保留。
- 范围：提供对外 API、核心实现、构建脚本、测试样例与文档体系；暂不包含线程安全、syslog、彩色输出与结构化日志（后续迭代）。
- 里程碑：
  - 2025-12-14：完成 v0.1 基础库、构建与测试、文档体系建立。

## 实施方案
- 技术栈：C（GNU 编译器）、GNU Make。
- 目录结构：
  - `include/logger.h` 对外 API
  - `src/logger.c` 核心实现（级别过滤、时间格式化、大小轮转）
  - `Makefile` 构建与测试目标
  - `tests/test_logger.c` 功能验证用例
  - `docs/日志模块/*` 6A 工作流文档（ALIGNMENT/CONSENSUS/DESIGN/TASK/ACCEPTANCE/FINAL/TODO）
- 构建与验证：
  - 构建静态库：`make lib`
  - 运行测试：`make test`
